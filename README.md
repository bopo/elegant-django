# Elegant Django

**针对 Django 管理界面的现代主题**

Elegant Django 是 [Django](http://www.djangoproject.com) 管理界面的替代主题/皮肤/扩展。

* 项目主页：https://github.com/bopo/elegant-django
* 在线演示：http://djangosuit.com/admin/
